# aula_git_pratica

Modificando remotamente pelo github

Modificando localmente pelo vscode 

Comandos - 

git pull - Puxa as modificações do Github para a máquina local

git status - Verifica se os arquivos do sistema batem com os do repositório remoto

git add (coloque o nome do arquivo aqui) - Adiciona o arquivo ao "pacote"

git commit -m(mensagem) - Envia o "pacote" ao github atualizado

git push - Envia o commit ao github
